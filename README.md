QUICK START (takes 10 seconds)
-------------------------------
  1. Unzip this folder anywhere on your computer
  2. Double-click  Animated-Weekdays-Live-Wallpaper.html
  3. It opens in your browser — that's it, you're done!

  ✅ Works 100% offline after the first open
     (fonts load from Google Fonts on first launch — needs internet once)

  Recommended browsers: Chrome, Firefox, or Edge
  Works on: Windows 10/11 and macOS

FEATURES
─────────────────────────────────────────
  • Displays the current day of the week, full-screen
  • Breathing / pulse animation (can be toggled off)
  • Live clock & date shown top-right
  • 7 languages: English, French, German, Spanish,
                 Japanese, Persian (Farsi), Hebrew
  • 5 font choices: Inter, Poppins, Montserrat,
                    Rubik Mono, Monoton

  • Day preview — click any day to preview it
  • Click the canvas to pause / resume animation
  • Fullscreen button (bottom-right)
  • Your settings are remembered between sessions

─────────────────────────────────────────
HOW TO USE IT AS A DESKTOP WALLPAPER
─────────────────────────────────────────

  WINDOWS — Lively Wallpaper (free, from Microsoft Store)
  --------------------------------------------------------
  1. Download "Lively Wallpaper" from the Microsoft Store (it's free)
  2. Open Lively Wallpaper
  3. Click the  +  button (Add Wallpaper)
  4. Drag and drop  Animated-Weekdays-Live-Wallpaper.html  into the window
  5. Done — it runs live on your desktop!

  MAC — Plash (free, from the Mac App Store)
  -------------------------------------------
  1. Download "Plash" from the Mac App Store (it's free)
  2. Open Plash → click the menu bar icon → Preferences
  3. Click the folder icon and select  Animated-Weekdays-Live-Wallpaper.html
  4. Done — it runs live on your desktop!

  JUST BROWSER FULLSCREEN (Windows & Mac)
  ----------------------------------------
  1. Open the file in Chrome or Firefox
  2. Press  F11  (Windows)  or  Cmd+Shift+F  (Mac)  to go fullscreen
  3. Great for a second monitor, a dedicated display, or a kiosk screen

─────────────────────────────────────────
TROUBLESHOOTING
─────────────────────────────────────────
  Fonts not loading / showing boxes?
    → You need an internet connection for the very first open.
      After that it works offline.

  Animation looks choppy or slow?
    → Make sure Hardware Acceleration is ON in your browser:
      Chrome: Settings → System → "Use hardware acceleration when available"
      Firefox: Settings → Performance → "Use hardware acceleration"

  Colours reset after re-opening?
    → Make sure you're opening the file the same way each time
      (same browser, same file path). Settings are saved per-browser.

  File won't open on Mac?
    → Right-click the file → Open With → choose your browser.
